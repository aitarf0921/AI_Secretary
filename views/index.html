Customize Your AI Chatbot Parameters for Seamless Integration

Attribute Required Description
src Required Load the core script for your AI chatbot embedding.
data-position Optional Position the floating button—default: bottom-right for optimal visibility.
data-site Required Your unique site ID for knowledge base integration.
data-accent Optional Brand theme color (HEX/RGB) to match your site's design.
data-width Optional Chat panel width (px), minimum 240 for responsive display.
data-height Optional Panel height (px), minimum 320; auto-adjusts on mobile devices.
data-z Optional Z-index to ensure your AI chatbot stays on top without overlap.
